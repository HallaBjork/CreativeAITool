INFORMATION:

There is no one right way to use this creative application. The Step-by-step guide is there only for inspiration on how to use the model. The idea is for the user to decide how much and in what way they want the model to be a part of their creative process. This application will not necessarily give you solutions to your creative problems but rather inspiration and insight into options that you might not have thought of. It is all up to you how you interpret the results and the way you choose to use the model.

STEP-BY-STEP GUIDE:

Step1: Draw an image using the brushes from the drop down in the control panel.

Step2: Save the image and it will appear in GALLERY tap under EVOLVE button.

Step3: Push the EVOLVE button to slide open a panel and choose an image from the GALLERY tab.

Step4: After selecting an image push the TO MODEL button and go to the MODEL tab.

Step5: Use the options as you like to change the models parameters.

Step6: Choose 1-5 images (parents) to evolve the next generation and push the button BREED.

Step7: Repeat steps 5 and 6 as often as you see fit.

Step8: If you see an image that you like you can save it by pushing the SAVE button.

Step9: You can then go to the GALLEY tab and further work on an image by pushing TO BOARD.

Step10: Close the sliding panel and now you can work on the new image and repeat all the steps.
