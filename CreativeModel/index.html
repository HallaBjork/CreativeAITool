INFORMATION:

This application combines CPPNs evolutionary images and traditional drawing methods. The guide is there to describe the functions of the model but you choose how you use it and how you interpret its results.

GUIDE:

MODEL Open the model-tab at the top of this panel.

i You can choose 1-5 images (parents) to generate the next generation of images.

EVOLVE To evolve the next generation, push the evolve-button.

RESTART To start a new image selection, push the restart-button and repeat steps above.

i If you want to work on an image, move it to the drawing board by double clicking it.

i Information about the selected image is to the right of the model.

i You can draw using the brushes from the drop down in the control panel on the top left.

SAVE To save your work push the save-button in the control panel on the top left.

CLEAR To clear the drawing board push the clear-button in the control panel on the top left.

i For the best viewing experience use Google Chrome.
